Conclusion
-------------

Analysis of RNA-Seq data using Kallisto and Sleuth is a fast an accurate way to
quantify gene expression (transcript abundance). Deploying this analysis on
CyVerse allows you to use these tools without expert knowledge of the command
line and  adds numerous features for experimental reproducibility as well as
data management and sharing.
